AI编程工具学习

发表于 2026-01-20
本文字数： 661 阅读时长 ≈ 1 分钟

记录一下在日常学习中看到的AI编程工具排名，之后自己使用了再来看是不是这样。2025/12/30

学习自UPAI编程工具实战排名，谁是最好用的AI编程工具_哔哩哔哩_bilibili

首先是分了四个档次，A,A+,S-S。首先来看A的

一、A级

1.CODE BUDDY

高级程度，非常全面，小程序生态开发继承很好

2.KIRO

擅长从0到1的开发，但不适合做1-N的开发；他的SPEC对话方式比较重。应该是说这种修改量大

3.OPENCODE

开源的，命令行开发工具

二、A+级

1.windsurf

不温不火，高性价比，15美金，自己有SWE模型

2.DROIO

也有SPEC模式，只会出文档，但不改代码，支持SUBAGENT，交互界面最清晰

3.TRAE

Claude模型断供，存在多种功能。整个我使用过，效果确实很好，有Gemini3.0支持；不过普通模式效果不是很好。消耗的请求速度很快

4.Warp

也是终端命令行工具，能保存所有的终端命令

三、S-级

1.cursor

生态做的好，是引领者

2.augment code

目前做大项目表现最好的AI工具，主要是代码索引能力强，就是Token消耗太快

3.google antigravity

windsurf里面迭代出来的，浏览器debug的使用有意思，被强烈推荐

4.OPENAI CODEX

功能不是很好用，但是基础模型能力强。工具和交互差，但是GPT泰国好用

四、S级

1.claude code都很强，skii这样的工作流全部转化成他的skills了。只要模型支持ANthropic的协议
