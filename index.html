二、异或的性质

异或是位运算中的重点操作，满足以下定律：

  1. 归零律：x ^ x = 0 (任何数与自己异或都是0)
  2. 恒等律：x ^ 0 = x (任何数与0异或保持不变)
  3. 交换律与结合律：a ^ b ^ c = a ^ c ^ b

三、 必背的高阶技巧

  1. 消除二进制中最低位的 1 (Brian Kernighan 算法)

  • 公式：n & (n - 1)
  • 原理：n-1 会把 n 最右边的 1 变成 0，并将该位右边的所有 0 变成 1。再和 n 做与运算，那部分就全部抵消了。
  • 应用：统计 1 的个数、判断是否为 2 的幂。

Java

2. 获取最低位的 1 (Lowbit)

  • 公式：n & -n
  • 原理：利用补码特性，-n 实际上是 ~n + 1。
  • 示例：x = 10100 -> lowbit = 00100
  • 应用：树状数组（Fenwick Tree）、Diff 比较。

  1. 判断 n 是否是 2 的幂

  • 公式：n > 0 && (n & (n - 1)) == 0
  • 原理：2 的幂次在二进制中只有一个 1。消掉这个 1 后应该等于 0。
