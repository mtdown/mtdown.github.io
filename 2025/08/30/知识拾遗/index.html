1.二维数组里面可以套不同类型数据吗？

不可以。Java 是一种静态类型语言，这意味着一个变量或数组在声明时，它的类型就已经被严格固定了,当你声明 int[][] x 时，你告诉编译器：“x 是一个数组，它的每个元素是另一个数组，而那个数组里的每个元素必须是 int 类型。

2.数组初始化的数值是多少？

是0，此外，这是各种基本类型的数组默认值：

数据类型 默认值
int, long, short, byte ``****0
float, double 0.0
char ```’\u0000’` (空字符)
boolean false
所有对象类型 (如 String, Integer 或其他类) null

3.对于一个二维数组int[][] x=new int[2][/2]，可以通过x[0][0]=new int[2]继续堆叠深度吗

不可以，这会导致编译错误，x的类型是是 int[][] (一个 int 数组 的 数组)。这是一开始就被决定的，这样做本质是将一个数组 (int[]) 赋值给一个只能存放单个整数 (int) 的位置。这在类型上是完全不匹配的。

4.List 和 Array 的核心区别

特性 List<List> (使用 ArrayList) String[][] (二维数组)
大小 (Size) 动态可变 (Dynamic) 固定不变 (Fixed)
声明与初始化 List<List> result = new ArrayList<>();
result.add(new ArrayList<>());		 String[][] result = new String[5][3];
说明 你可以随时 add 新的元素（新的列表），或从列表中 remove 元素。外层和内层列表的大小都可以随时改变。 在创建时必须指定大小 (new String[rows][cols])，之后长度无法改变。若要扩展，必须创建一个新数组并复制旧数据。
类型 **类 (Class)**，属于 Java 集合框架 (Collections Framework) 的一部分。 基础数据结构，是 Java 语言内置的一部分。
功能/方法 功能丰富。提供了大量便利的方法，如 add(), remove(), get(), size(), isEmpty(), contains(), sort() 等。 功能基础。只有一个 .length 属性来获取长度。所有操作（如新增、删除）都需要手动处理索引和数据复制。
性能 - 访问: get(i) 速度很快，为 O(1)。
- 新增/删除: 在末尾新增很快 (摊销 O(1))，但在中间插入/删除较慢 (O(n))。		 - 访问: array[i] 速度极快，为 O(1)。
- 新增/删除: 非常慢，因为需要创建新数组和复制数据 (O(n))。
泛型 (Generics) 必须使用泛型 () 来指定存储的对象类型。不能存储基本数据类型 (如 int)，需使用其包装类 (Integer)。 可以直接存储基本数据类型 (如 int[][]) 或对象类型 (如 String[][])。

6.String到基本类型的转换

指的注意的是，String作为我们常用的类型，它实际上不是基本类型，而应该算是包装类，对于除了char以外都有parseXXX这样的格式转化为对应的类型。

目标基本类型 使用方法 示例

long Long.parseLong(String s) long l = Long.parseLong(“1234567890”);
double Double.parseDouble(String s) double d = Double.parseDouble(“3.14”);
float Float.parseFloat(String s) float f = Float.parseFloat(“1.23f”);
boolean Boolean.parseBoolean(String s) boolean b = Boolean.parseBoolean(“true”);¹
byte Byte.parseByte(String s) byte by = Byte.parseByte(“120”);
short Short.parseShort(String s) short sh = Short.parseShort(“30000”);
char (特殊情况) char c = “hello”.charAt(0);²
